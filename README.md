# Harmonize 하모나이즈

**Harmonize**는 사용자의 음역대를 분석하여, 사용자들이 부를 만한 노래를 추천해주는 음악 추천 시스템입니다. 이 프로젝트는 안드로이드 앱과 스프링부트 REST API 서버, 파이썬/텐서플로 기반 음원 분석 프로그램으로 구성됩니다.
